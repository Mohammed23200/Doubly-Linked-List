Splite

the Split function divides the original chain into two halves: The first half contains nodes from the start of the chain up to (but not including) the midpoint. The second half contains nodes from the midpoint to the end of the chain. The function uses the "tortoise and hare" technique to find the midpoint efficiently, ensuring that the split is done in a single pass through the list. After the split, the firstHalf and secondHalf chains are populated with the respective nodes.
